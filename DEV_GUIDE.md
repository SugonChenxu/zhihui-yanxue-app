# 开发环境搭建指南

## 方案一：浏览器预览（最快，现在就能用）

### 步骤
1. 用 Chrome 浏览器打开 `app/index.html`
2. 按 `F12` 开启开发者工具
3. 点右上角的手机/平板图标（Toggle device toolbar）
4. 选择 iPhone 12 Pro 或 Pixel 5 模拟手机尺寸
5. 刷新页面，就能像 App 一样操作了

### 优点
- 无需安装任何软件
- 修改代码后刷新即看效果
- 可以模拟不同手机尺寸

---

## 方案二：HBuilderX 打包真机APK（参赛演示用）

### 安装步骤
1. 访问 https://www.dcloud.io/hbuilderx.html
2. 下载 Windows 版（免费）
3. 安装后打开，选择「新建项目」→「5+App」
4. 把 `app/` 目录下的文件拷贝到项目目录
5. 连接安卓手机，开启「开发者模式」→「USB调试」
6. 点「运行」→「运行到手机或模拟器」→ 选择你的手机

### 打包APK步骤
1. 在 HBuilderX 中打开项目
2. 点「发行」→「原生App-云打包」
3. 选择 Android，等待打包完成（约5-10分钟）
4. 下载APK文件，安装到手机

### 优点
- 生成真APK，可以在任何安卓手机上安装
- 中文界面，操作简单
- 免费云打包

---

## 方案三：Android Studio（完整开发环境，可选）

> 仅当需要深度定制原生功能时才需要，参赛demo阶段不推荐

### 系统要求
- Windows 10/11 64位
- 8GB RAM 以上（推荐16GB）
- 10GB 可用磁盘空间

### 安装步骤
1. 访问 https://developer.android.com/studio
2. 下载安装 Android Studio
3. 安装时勾选「Android Virtual Device」
4. 创建虚拟设备（Pixel 5 或类似）
5. 用 WebView 加载本地 HTML 文件
