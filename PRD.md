## 一、产品概述

### 1.1 产品名称
智绘研学e启航 课程APP

### 1.2 产品定位
面向大中小学生群体的公益红色教育慕课平台，依托天津红色旅游资源，通过线上微课+线下沉浸教学双重模式，厚植爱国主义情怀。

### 1.3 目标用户
| 用户群体 | 特点 |
|---------|------|
| 小学生 | 趣味教育+动手实践，红色绘本、文物拼图 |
| 初高中生 | 互动体验，角色扮演、红色剧本杀 |
| 大学生 | 思辨性+思考，红色思辨辩论、红色戏剧 |
| 留学生 | 双语教学、文化融合 |

### 1.4 核心亮点
- 🎯 区分受众：按学段设计不同课程内容
- 📱 数字赋能：VR体验、直播互动微课
- 🗺️ 模块设计：地标/人物/文物/事件四大课程模块
- 🎁 文创推广：红色文创产品展示

## 三、视觉设计规范

### 3.1 整体风格
**经典红色风** — 以正红+金色为主，庄重大气，突出红色教育的严肃感和仪式感。

### 3.2 配色方案
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#C91A1A` | 正红，用于导航栏、按钮、强调元素 |
| 辅色 | `#D4A017` | 金色，用于图标、分割线、装饰元素 |
| 背景 | `#FFF8F0` | 暖白，页面背景 |
| 卡片 | `#FFFFFF` | 纯白，内容卡片 |
| 文字主 | `#1A1A1A` | 深黑，主要文字 |
| 文字辅 | `#666666` | 灰色，辅助文字 |
| 成功色 | `#1D9E75` | 绿色，打卡/完成状态 |
| 警告色 | `#BA7517` | 橙色，提示信息 |

### 3.3 Logo设计
- 元素：天津地标（天津之眼/解放桥）+ 红色星形元素
- 色调：红金配色
- 字体：雅黑加粗，中文排版
- 文件：`app/assets/logo.png`

## 四、页面结构图

```
App
├── 启动页（Splash）
├── 登录/注册
├── 【首页】（底部Tab 1）
│   ├── 轮播图
│   ├── 课程分类入口（4个模块）
│   └── 推荐课程列表
├── 【课程】（底部Tab 2）
│   ├── 模块筛选标签
│   ├── 课程列表
│   └── 课程详情 → 视频播放
├── 【我的】（底部Tab 3）
│   ├── 用户信息
│   ├── 学习记录
│   ├── 我的收藏
│   ├── 关于我们
│   └── 设置
└── 【特色功能页】（从首页入口进入）
    ├── VR体验（概念页）
    ├── 直播微课（占位页）
    └── 课程地图
```

## 五、非功能需求
